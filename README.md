# 🚀 O Caminho do Arquiteto: Ciência da Computação Guiada por IA (2026)

Bem-vindo ao **Caminho do Arquiteto**, um currículo de Ciência da Computação open-source desenhado para a Era da Inteligência Artificial. 

Este projeto foi criado para levar uma pessoa com zero conhecimento em programação a se tornar um Arquiteto de Software de elite. A premissa fundamental deste curso é que a transição para a Era da IA não tornou os fundamentos obsoletos; pelo contrário, elevou o valor da compreensão profunda de sistemas, redes e algoritmos [1]. O papel do engenheiro moderno evoluiu da simples digitação de código para a orquestração de agentes e avaliação crítica de sistemas complexos [1, 2].

## 🧠 A Metodologia: O Tutor Socrático

O maior risco do estudo autodidata com IA é a "ilusão de competência" — achar que aprendeu algo só porque a máquina gerou a resposta certa [3]. 

Para evitar isso, este curso não usa a IA como um gerador de código, mas como um **Tutor Socrático** [4, 5]. A IA é configurada para:
- **Nunca fornecer soluções completas** de imediato [6, 7].
- Focar no **esforço produtivo**, deixando o aluno lutar com o problema para garantir a retenção do conhecimento [3, 8].
- Usar uma **escada de dicas graduais** (Níveis 0 a 4), indo de um leve apontamento de direção até correções guiadas, sem dar o código final [5, 9].
- Forçar o aluno a testar o código e validar o design através de um ciclo contínuo de Test-Driven Development (TDD) [10, 11].

## 🗺️ Roadmap de Estudos

O currículo é dividido em 12 módulos progressivos, onde cada etapa constrói a fundação para a próxima, utilizando o Linux (Ubuntu) como o laboratório principal do aluno [12].

* **Módulo 0: IA - Literacia e Engenharia de Prompt:** Aprendendo a usar a máquina. Frameworks como CRAFT e CO-STAR, e a mitigação de alucinações [13-15].
* **Módulo 1: O Ambiente do Engenheiro e Lógica:** Instalação do Linux Ubuntu, uso do terminal (CLI), Shell Scripting, Git e os fundamentos da programação [16].
* **Módulo 2: Arquitetura de Computadores:** Portas lógicas, CPU, memória RAM, Assembly e inspeção de hardware nativa no Linux (`lscpu`, `free -m`) [12, 17].
* **Módulo 3: Matemática para Computação:** Lógica booleana, Álgebra Linear (a base da IA) e Estatística [18, 19].
* **Módulo 4: Estruturas de Dados:** Arrays, Listas Encadeadas, Tabelas Hash, Árvores e Grafos, com foco em otimização de memória [20, 21].
* **Módulo 5: Algoritmos:** Busca, ordenação, notação Big-O, programação dinâmica e algoritmos gulosos [21, 22].
* **Módulo 6: Sistemas Operacionais:** Processos, Threads, gerenciamento de memória virtual e o sistema de arquivos do Linux (Ubuntu como laboratório de estudo) [23].
* **Módulo 7: Redes de Computadores:** O Modelo OSI, protocolos TCP/UDP, DNS, HTTP e uso de ferramentas de rede do Linux (`curl`, `netstat`) [1].
* **Módulo 8: Bancos de Dados:** SQL vs NoSQL, modelagem relacional, transações ACID e Bancos de Dados Vetoriais para IA [24, 25].
* **Módulo 9: Qualidade, Segurança e Testes:** O novo ciclo AI-SDLC, TDD com agentes, DevSecOps e prevenção de vulnerabilidades e injeção de prompt [26-28].
* **Módulo 10: Sistemas Distribuídos:** Teorema CAP, Microsserviços, Mensageria, Containerização (Docker/Kubernetes no Ubuntu) e observabilidade causal [29, 30].
* **Módulo 11: Projeto Final (Portfólio):** Construção de sistemas nativos de IA, orquestração de múltiplos agentes e deploy na nuvem [31, 32].

## 🛠️ Projetos Práticos e Portfólio (Pronto para 2026)

Os recrutadores de hoje não querem ver tutoriais copiados. Eles buscam evidências de sistemas implantados que resolvam problemas reais [32, 33]. Durante o curso, você construirá projetos de nível de produção, tais como:

1. **Sistema RAG com Controle de Acesso (RBAC):** Um sistema que consulta bases de conhecimento (PDFs/Manuais) usando LLMs, garantindo que usuários só acessem dados permitidos pelo seu nível de segurança [32, 34].
2. **Assistente de Programação Multi-Agente:** Um sistema colaborativo onde um agente planeja a arquitetura, outro escreve o código, um terceiro revisa a segurança e um quarto roda os testes automatizados [25, 35].
3. **Análise Financeira Multimodal:** Uma aplicação que processa relatórios em PDF, gráficos em imagem e dados tabulares ao mesmo tempo, gerando resumos analíticos com proteção rigorosa contra alucinações de IA [32, 36].
